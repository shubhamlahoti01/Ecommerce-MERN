# A Full Stack Ecommerce Website Where you can Register/Login, Create Product(Admin Panel), AddToCart Feature, Checkout with Stripe Payment Gateway, Pagination and Filters for products and etc.
## -Tools & Technologies used:- React, Redux, Node, Express, MongoDB, Axios
## -Learnings: Creating REST API for the Ecommerce Website features, Designing Database Model in MongoDB, Implementing Authentication and Validations with Password Encryption using BcryptJs, Pagination of Products, Sorting and Filtering of products as per choice, Categories Selection and many more.
